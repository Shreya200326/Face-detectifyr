Face Detectifyr

Face Detectifyr is a robust face detection model leveraging OpenCV and machine learning techniques to accurately detect and localize faces in images or video streams. It's ideal for real-time applications and can be easily integrated into various AI projects.

Features

  • Accurate face detection using OpenCV.
  • Real-time processing for video streams.
  • Customizable parameters for different use cases.
  • Easy integration into AI and computer vision workflows.

Installation

Follow the steps below to set up the project:

git clone https://github.com/ayushi-shahi/Face-Detectifyr.git
cd Face-Detectifyr
pip install -r requirements.txt

Usage

Run the following command to start face detection:

python face_detectifyr.py

Ensure your input images or video streams are in the correct directory or specify the path when running the script.
